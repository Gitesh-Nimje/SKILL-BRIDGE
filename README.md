# Skill Gap Analysis & Personalized Career Roadmap Platform

##  Overview
This platform analyzes a user's current skills, compares them with industry-required skills, and generates a personalized career roadmap.

##  Features
- Skill Gap Analysis
- Skill Match Percentage
- Missing Skills Identification
- Personalized Learning Roadmap
- Progress Tracking Dashboard

##  Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js / Flask
Database: MongoDB / JSON

##  How It Works
1. User selects career domain.
2. User enters current skills.
3. System compares with industry requirements.
4. Platform calculates skill match percentage.
5. Personalized roadmap is generated.
